# Mood Line

Turn a user's described feelings, mood, or emotional state into a concise, resonant line — accurate, restrained, and human.

## What it does
- Identifies the core emotion and its underlying source from the user's raw description
- Provides a natural-language emotion interpretation without psych-diagnosis or platitudes
- Outputs an expressive line, using verified quotes with proper attribution only when reliable
- Offers optional variants tuned for social media, diary, sharing with friends, or self-reflection

## Example scenarios

**1. 努力未被看见后的委屈**
> 👤 "最近很努力，但总感觉没人看见，心里有点委屈，又觉得是不是自己太矫情了"
> 🤖 情绪核心：努力未被看见后的委屈 + 自我怀疑。情绪解读：你不是矫情，而是在持续付出后，期待被理解却迟迟没有等到。表达句：你难过的不是辛苦本身，而是那些没有回声的认真。

**2. 刚结束一段关系，想要克制的话发朋友圈**
> 👤 "刚结束一段关系，想要一句克制一点的话发朋友圈"
> 🤖 情绪核心：失落 + 慢慢接受现实的克制。情绪解读：你不是不在乎了，只是在学着把舍不得收回心里。表达句：有些告别并不喧哗，它只是把一个人的位置，慢慢还给沉默。

**3. 关于孤独的名言（引用真实性校验）**
> 👤 "给我一句关于孤独的名言"
> 🤖 先检查白名单中是否有可可靠归因的孤独相关引用。如果有，输出引用金句并标明来源。如果没有可靠出处，输出自然的无署名表达，不伪造名言或使用模糊归属。
